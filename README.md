# 🧮 CLI Calculator

Ein modulares Python-Kommandozeilen-Tool (CLI) für mathematische, trigonometrische und geometrische Berechnungen.

## ✨ Features
* **Grundrechenarten:** `+`, `-`, `*`, `/`
* **Wissenschaftlich:** Potenzieren (`^`), Quadratwurzel (`sqrt`), Fakultät (`!`)
* **Trigonometrie:** `sin`, `cos`, `tan` (in Grad) & Inversa (`asin`, `acos`, `atan`)
* **Geometrie:** Umfang & Fläche von Kreisen, Flächeninhalt von Rechtecken
* **Robust:** Integrierte Fehlerbehandlung für Division durch Null oder negative Radien/Wurzeln
* **File:** Mit einem eigenen History File

## 🚀 Ausführung
Starte das Skript einfach in deinem Terminal:
```bash
python main_calc.py
